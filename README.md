# 🏠 Housing Market Trends: An Analysis of Sale Prices and Features using Tableau

An interactive Tableau dashboard project that analyzes housing market trends to uncover insights into property prices, house features, and renovation impacts.

---

## 📌 Project Overview

This project focuses on analyzing residential housing data using Tableau to help users understand:

- Sale price trends
- Renovation impact on prices
- House age distribution
- Bedroom & bathroom analysis
- Basement area analysis
- Interactive dashboards and stories

---
